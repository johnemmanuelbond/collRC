render module
=============

Links together methods from :doc:`calc <calc>`, :doc:`coloring <coloring>`, and :doc:`visuals <visuals>` to process simulation data for visualization. Also contains color base classes and utilities for defining color schemes based on reaction coordinates. Mostly concerned with plotting single `matplotlib <https://matplotlib.org/stable/contents.html>`_ figures to render using `moviepy <https://zulko.github.io/moviepy/>`_.
